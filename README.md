# Icey
A simple python program for screen pet.

Pet is inspired from icey of "We bear bears".

![63455](https://user-images.githubusercontent.com/67326784/121695364-9ff83780-cae8-11eb-9fec-b50c62ab6a81.jpg)

# Modules used:
```
* tkinter 
* pygame
* typing
```
